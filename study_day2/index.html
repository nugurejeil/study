2. HTML 요소의 종류2

  • <P>

    : paragraph. 문장의 의미하는 요소입니다.

  • <hr>

    : 콘텐츠 사이의 가로 구분선입니다. —> 위 아래 콘텐츠의 내용이 다르거나, 내용의 테마를 바꾸고자 할 때 사용합니다.

    <h1> 태그의 경우 <section> 태그안에 들어갈 경우 폰트 사이즈가 바뀝니다.** —> section안에 들어가는 테그는 h2부터가 적당한듯합니다.

  • <pre>

    : html에 작성되어 있는 내용을 그대로 보여줍니다. 따로 편집할 필요가 없어 편해보이긴 하지만 태그에 ‘아무런 의미가 없기’ 때문에 주로 간단한 수식이나 코드를 보여주고자 할 때만 사용합니다.

  • <blockquote>

    : 인용해오는 구절 혹은 컨텐츠를 의미하는 태그입니다. 저작권 관련 문제 때문에 사용합니다. quote: 누군가 한 말(가라사대, 왈)

  • <ol>

    : 순서있는 목록을 나타내는 요소입니다. 직계자식 요소로 반드시 <li>요소를 사용해야 합니다.

  • <ul>

    : 순서가 없는 목록입니다. 직계자식 요소로 반드시 <li>요소를 사용해야 합니다.

  • <li>

    : 리스트에서 사용하는 아이템입니다. 부모 요소로 반드시 <ol> 혹은 <ul> 을 가져야 합니다.

  • <dl>, <dt>, <dd>

    : 주로 사전이나 검색결과 등 특정 대상과 그 설명을 해야할 경우 사용해야 하는 요소입니다. 무조건 <dt> 부터 시작하여 <dd> 로 이어져야합니다. —> 작은 단위의 컨텐츠에 주로 사용합니다. <h1> 과 같은 heading 요소를 사용하기가 애매할 경우 사용합니다. 

                        <body>
                        <dl>
                            <dt>치킨</dt>
                            <dd>치킨은 닭튀김이다.</dd>
                            <dd>치킨은 진리다.</dd>
                        </dl>
                    </body>
  • <figure>, <figcaption>

    : 특정 문서에서 컨텐츠를 부연 설명 하기 위한 컨텐츠를 나타냅니다. 주로 이미지와 함께 사용합니다. <aside> 와 다른점은 좀 더 작은 단위의 컨텐츠에 사용한다는 것입니다.

                        <figure>
                        <img src="나의 사진.png" alt="">
                        <figcaption>내가 16살때, 성산일출봉 앞에서 찍은 사진이다.</figcaption>
                    </figure>
  • <div>

    : 특정한 의미 없이 그룹을 묶어 구획화시키기 위해 사용. 의미가 없다보니 class와 같이 사용해줍니다. 마땅히 시멘틱한 표현의 셀렉터를 사용할 수 없을 경우! 사용해요 ^^

  • <main>

    : <body> 요소 안의 메인 컨텐츠를 나타내기 위해 사용합니다. HTML문서안에 반드시 하나만 사용합니다.

  • <table>

    : 테이블 요소입니다. table 요소로 페이지의 레이아웃을 작성하는건 지양해야 합니다. 의미도 맞지 않고 테이블 안에 테이블 요소를 넣게 되는 일이 빈번한데 그렇게 구조가 깊어지면 성능저하가 발생합니다. 하지만 컨텐츠 내에서 테이블을 작성해야 할때는 사용합니다!

  • <caption>

    : 테이블의 제목입니다. —> 권고사항 : “The <caption> tag must be inserted immediately after the <table> tag.”(table태그 바로 뒤에 caption 태그를 넣어 주세요.)

  • <thead>, <tbody>, <tfoot>

    : 테이블의 헤드, 바디, 푸터 부분을 의미합니다. 모두 선택적 입력사항이지만 바디는 반드시 작성해주는게 좋습니다.

    • <tr> : 테이블의 한줄을 의미합니다. (table row)

    • <th> : 테이블의 내용 중 제목에 해당하는 내용을 의미합니다. (table header)

    • <td> : 테이블의 내용 중 데이터에 해당하는 내용을 의미합니다. (table data) 

                              <table>
                            <caption>
                              철수의 모든것
                            </caption>
                            <tfoot>
                                <tr>
                                    <td>결론</td>
                                    <td colspan="3">완벽하게 멋져</td>
                                </tr>
                            </tfoot>
                            <tbody>
                              <tr>
                                <th>이름</th>
                                <th>나이</th>
                                <th>전화번호</th>
                              </tr>
                              <tr>
                                <td>철수</td>
                                <td>60</td>
                                <td>0000-0000</td>
                              </tr>
                            </tbody>
                        </table>

      **<tfoot> 태그의 경우 <tbody> 보다 앞에 써도 보여지는 결과는 가장 뒤에 따라 옵니다. 시각장애인의 편의성을 위해 결과를 먼저 보여주도록 만들었습니다.

      colspan =“ ” : column을 병합합니다.(span의 의미: 둘 사이를 잇다.)

  • <form>요소들
    • <input>

      : 데이터를 입력받아 서버에 전달하는 요소입니다. type의 속성에 따라 다양한 일을 수행합니다.

                              <form>
                            <input type="text">
                            <input type="password">
                        </form>
    • <label>

      : 입력폼에서 받을 데이터를 소개합니다.

                              <label for="man">남자</label>
                        <input type="radio" id="man" name="gender">
                        <label for="women">여자</label>
                        <input type="radio" id="women" name="gender">
    • <button>

      : 유저가 클릭 했을 때 무언가를 행할 수 있는 버튼입니다.

    • <textarea>

      : 여러줄로 텍스트를 받고 싶을 때 사용합니다. 그리고 내용물에 따라 크기 조절이 가능합니다.

Text Level HTML 요소들

  • <a>

    : 페이지간의 링크를 정의합니다. (anchor)

  • <em>

    : 약한 강조입니다.

  • <strong>

    : 강한 강조입니다.

  • <abbr>

    : 약어를 나타냅니다.

    <abbr title="World Wide Web">www</abbr>
  • <q>

    : 한줄 인용구입니다.

  • <cite>

    : 출처를 나타냅니다.

  • <dfn>

    : 문서에서 최초로 정의한 단어를 나타냅니다. 보통 뒤에 단어의 설명이 따라오는 구조입니다. (define tag)

    <p><dfn>치맥</dfn> 은 치킨과 맥주를 합친 신조어입니다.</p>
  • <time>

    : 시간을 나타냅니다.

                            <time>10:00</time>
                        <time datetime="2008-02-14 20:00">Valentines day</time>
  • <code>

    : 문서상에서 코드를 명시적으로 나타내는 요소입니다.

  • <span>

    : 자체의 의미는 없으나 다양한 속성을 부여하기 위해 사용합니다. (div 와 비슷하지만 div가 봉지라면 span 스티커 같은 느낌)

  • <br>

    : 줄바꿈 요소입니다.

  • <img src="" alt="" title="">

    : 이미지 요소입니다. alt는 이미지 로딩 실패 시 대체할 수 있는 텍스트를 넣어줍니다. title은 이미지의 제목 혹은 주제를 나타냅니다.

  • <del>

    : 문서상에서 삭제된 내용을 말합니다.

  • <s>

    : 문서상에서 더 이상 적절하지 않은 내용을 말합니다.
